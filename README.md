## ส่วนที่ 4: วิธีติดตั้ง Python บน Windows

### 1) เข้าเว็บดาวน์โหลด Python

ให้เปิดเว็บทางการของ Python:

[https://www.python.org/downloads/](https://www.python.org/downloads/)

เมื่อเข้าไปแล้ว จะเห็นหน้าดาวน์โหลดหลัก

---

### 2) วิธีดาวน์โหลด

ในหน้า Downloads ให้มองหาปุ่มสำหรับ Windows แล้วกดดาวน์โหลดตามที่หน้าเว็บแสดง

โดยทั่วไปสามารถกดปุ่มประมาณนี้:

- **Download Python install manager**

หรือถ้าต้องการไฟล์ติดตั้งแบบ `.exe` โดยตรง ให้เข้าไปที่:

[https://www.python.org/downloads/windows/](https://www.python.org/downloads/windows/)

แล้วเลือก:

- **Download Windows installer (64-bit)**

### 3) วิธีติดตั้ง Python

หลังจากดาวน์โหลดไฟล์เสร็จแล้ว ให้ทำตามนี้

- ไปที่โฟลเดอร์ **Downloads**
- ดับเบิลคลิกไฟล์ที่ดาวน์โหลดมา
- ถ้ามีหน้าติดตั้งเด้งขึ้นมา ให้กด **Install**
- รอจนติดตั้งเสร็จ

เมื่อติดตั้งเสร็จแล้ว ให้ปิดหน้าต่างติดตั้งได้เลย

### 4) วิธีเช็กว่าเครื่องมี Python อยู่แล้วหรือยัง

ทำการเปิด Command Prompt

- กดปุ่ม **Start**
- พิมพ์คำว่า **cmd**
- กดเข้า **Command Prompt**

จากนั้นพิมพ์คำสั่งนี้

```bash
python --version
```

หรือใช้คำสั่ง

```bash
py--version
```

ถ้าติดตั้งแล้ว จะปรากฏดังภาพ เช่น

```bash
Python 3.13.3
```

## ส่วนที่ 5: เช็กว่า pip ติดตั้งแล้วหรือยัง

pip คือเครื่องมือสำหรับติดตั้งไลบรารีของ Python ให้พิมพ์คำสั่งนี้ใน Command Prompt

```bash
pip --version
```

ถ้าขึ้นข้อความดังใต้ภาพเเสดงว่าใช้งานได้เเล้ว

```bash
pip 25.x from ...
```

## ส่วนที่ 6: ติดตั้งโปรเจ็คลงบนเครื่อง

### 1) ดาวน์โหลดไฟล์ ZIP จาก GitHub

- เข้าไปที่หน้าโปรเจกต์บน GitHub
- กดปุ่ม **Code**
- เลือก **Download ZIP**
- รอให้ดาวน์โหลดเสร็จ

### 2) แตกไฟล์ ZIP

- ไปที่โฟลเดอร์ **Downloads**
- คลิกขวาที่ไฟล์ ZIP
- เลือก **Extract All...**
- เลือกตำแหน่งที่ต้องการแตกไฟล์
- กด **Extract**

### 3) เปิดโปรเจกต์ใน VS Code

- เปิดโปรแกรม **Visual Studio Code**
- กด **File > Open Folder**
- เลือกโฟลเดอร์โปรเจกต์ที่แตกไฟล์ไว้
- กด **Select Folder**

### 4) เปิด Terminal ใน VS Code
- สังเกตุเเถบเมนูด้านบนจะพบคำว่า **Terminal**
- ทำการคลิกเเละเลือก **New Terminal**

### 5) ติดตั้ง Library พื้นฐาน สำหรับเชื่อมต่อฐานข้อมูล

โดยใช้คำสั่งดังนี้

```bash
pip install django mssql-django django-crispy-forms crispy-bootstrap5 openpyxl
```

หรือให้ error ให้ใช้คำสั่งนี้เเทน

```bash
python -m pip install django mssql-django django-crispy-forms crispy-bootstrap5 openpyxl
```
รอจนติดตั้งเสร็จ

### 8) ตั้งค่าการเชื่อมต่อใน `settings.py` ในโฟลเดอร์โปรเจ็ค
ไปที่ `Laundry_Project-main>>laundry_project>>laundry_project>>settings.py` แก้ไขส่วน `DATABASES` ให้ตรงกับเครื่อง:

```python
DATABASES = {
    'default': {
        'ENGINE': 'mssql',
        'NAME': 'LaundryDB',
        'USER': 'sa',
        'PASSWORD': '123456789',
        'HOST': 'ใส่ชื่อ device ตัวเอง เช่น DESKTOP-6445F79\\SQLEXPRESS2019',  
        'PORT': '',
        'OPTIONS': {
            'driver': 'ODBC Driver 17 for SQL Server',
            'extra_params': 'TrustServerCertificate=yes;',
        },
    }
}
```

- โดยสังเกตุ ชื่อ device จากภาพที่เราได้เข้า SSMS ใน **settings.py** ต้องใส่ \ 2 อัน <br><br>
